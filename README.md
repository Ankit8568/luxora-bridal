# Luxora Bridal ✨

Luxora Bridal is a modern and responsive bridal makeup website designed to showcase professional bridal beauty services, makeup packages, portfolios, and client experiences.

## 🌐 Live Website

Live Demo: https://luxora-bridal-x18l.vercel.app
## ✨ Features

- Modern and elegant bridal design
- Fully responsive layout
- Smooth animations and transitions
- Bridal makeup service showcase
- Portfolio and gallery sections
- Mobile-friendly navigation
- Contact and booking sections

## 🛠️ Technologies Used

- Next.js
- React.js
- TypeScript
- Tailwind CSS
- Framer Motion

## 📂 Project Structure

- `app/` – Website pages and layouts
- `components/` – Reusable website components
- `public/` – Images and other static assets
